# Othello
Task Management with Kanban/Scrum Boards

### Technologies : 
- **Client:** Angular,
- **REST API:** RubyOnRails

> **Source for Backend:** [Othello-backend](https://github.com/Suyious/Othello-Backend.git)

### Setup:
```bash
git clone https://github.com/Suyious/Othello.git
cd Othello
npm install
npm start
```
